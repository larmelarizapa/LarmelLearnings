Operators in Java

Description: Operators are special symbols used to perform operations on variables and values. Java provides a wide range of operators for various tasks, including arithmetic, relational, logical, and others. Understanding operators is essential to write efficient Java code.

Arithmetic Operators

Description: Arithmetic operators are used to perform basic mathematical operations like addition, subtraction, multiplication, division, and modulus (remainder).


  // Addition
  int a = 10;
  int b = 5;
  int sum = a + b;  // Result: 15
  
  // Subtraction
  int difference = a - b;  // Result: 5
  
  // Multiplication
  int product = a * b;  // Result: 50
  
  // Division
  int quotient = a / b;  // Result: 2 (integer division)
  
  // Modulus (remainder)
  int remainder = a % b;  // Result: 0

Relational Operators

Description: Relational operators are used to compare two values and determine the relationship between them. They return a boolean result: true or false.


  // Equal to
  boolean equal = (a == b);  // Result: false
  
  // Not equal to
  boolean notEqual = (a != b);  // Result: true
  
  // Greater than
  boolean greaterThan = (a > b);  // Result: true
  
  // Less than
  boolean lessThan = (a < b);  // Result: false
  
  // Greater than or equal to
  boolean greaterThanEqual = (a >= b);  // Result: true
  
  // Less than or equal to
  boolean lessThanEqual = (a <= b);  // Result: false

Logical Operators

Description: Logical operators are used to combine multiple boolean expressions. The result is a boolean value that indicates whether the combined condition is true or false.


  // Logical AND (&&): Returns true if both conditions are true
  boolean and = (a > b && b < 10);  // Result: true
  
  // Logical OR (||): Returns true if at least one condition is true
  boolean or = (a > b || b > 10);  // Result: true
  
  // Logical NOT (!): Inverts the boolean value
  boolean not = !(a > b);  // Result: false

Assignment Operators

Description: Assignment operators are used to assign values to variables. The most common assignment operator is =, but there are also shorthand versions for combined operations.


  // Simple assignment
  int x = 5;
  
  // Addition assignment
  x += 3;  // Equivalent to x = x + 3; Result: 8
  
  // Subtraction assignment
  x -= 2;  // Equivalent to x = x - 2; Result: 6
  
  // Multiplication assignment
  x *= 4;  // Equivalent to x = x * 4; Result: 24
  
  // Division assignment
  x /= 6;  // Equivalent to x = x / 6; Result: 4
  
  // Modulus assignment
  x %= 3;  // Equivalent to x = x % 3; Result: 1

Unary Operators

Description: Unary operators operate on a single operand. They include increment (++) and decrement (--) operators, as well as the negation operator (-) and logical NOT (!).


  // Increment operator
  int count = 5;
  count++;  // Result: 6 (post-increment)
  
  // Decrement operator
  count--;  // Result: 5 (post-decrement)
  
  // Unary negation
  int neg = -count;  // Result: -5
  
  // Logical NOT
  boolean flag = true;
  flag = !flag;  // Result: false

Ternary Operator

Description: The ternary operator is a shorthand for an if-else statement. It evaluates a condition and returns one of two values based on whether the condition is true or false.


  // Syntax: condition ? value_if_true : value_if_false;
  int max = (a > b) ? a : b;  // If a > b, max = a; otherwise, max = b
  System.out.println(max);  // Output: 10

Instanceof Operator

Description: The instanceof operator is used to test whether an object is an instance of a specific class or implements an interface.


  // Example using instanceof
  String str = "Hello, World!";
  boolean isString = str instanceof String;  // Result: true
