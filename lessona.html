Introduction to Java

Description: Java is a high-level, class-based, and object-oriented programming language. It is designed to have as few implementation dependencies as possible, making it platform-independent. Java follows the "Write Once, Run Anywhere" principle, meaning that once you write a Java program, it can run on any platform that has a Java Virtual Machine (JVM).

Setting Up Java Development Environment

Description: Before you start writing Java programs, you need to install the Java Development Kit (JDK). You will also need an Integrated Development Environment (IDE) to write and run your Java programs, such as IntelliJ IDEA, Eclipse, or NetBeans.


  // Install JDK from Oracle's website or use openjdk
  // Download an IDE like IntelliJ IDEA or Eclipse for coding

Java Syntax and Structure

Description: Java programs consist of classes, methods, and statements. The basic structure of a Java program includes defining a class, a main method, and writing statements inside the method.


  // A basic Java program structure
  public class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!"); // Output a message to the console
      }
  }

Variables and Data Types

Description: In Java, variables are used to store data. The data type of a variable determines the type of data it can store. Common data types in Java include int, double, boolean, char, and String.


  // Example of variable declaration and initialization
  int age = 19;
  double height = 5.9;
  boolean isStudent = true;
  char grade = 'A';
  String name = "Larmel";
  
  // Displaying values
  System.out.println("Name: " + name);
  System.out.println("Age: " + age);
  System.out.println("Height: " + height);
  System.out.println("Is Student: " + isStudent);
  System.out.println("Grade: " + grade);

Operators in Java

Description: Operators in Java are used to perform operations on variables and values. There are different types of operators in Java, such as arithmetic operators, comparison operators, logical operators, and assignment operators.


  // Arithmetic operators
  int sum = 5 + 3;
  int diff = 5 - 3;
  int product = 5 * 3;
  int quotient = 5 / 3;
  int remainder = 5 % 3;
  
  // Comparison operators
  boolean isEqual = (5 == 3);  // false
  boolean isGreaterThan = (5 > 3);  // true
  
  // Logical operators
  boolean and = (true && false);  // false
  boolean or = (true || false);  // true

Control Flow Statements

Description: Java provides control flow statements to control the execution of code based on certain conditions. These include conditional statements like if, else, and switch, as well as loop statements like for, while, and do-while.


  // Conditional statement
  int num = 10;
  if (num > 5) {
      System.out.println("The number is greater than 5");
  } else {
      System.out.println("The number is less than or equal to 5");
  }
  
  // Loop statement
  for (int i = 1; i <= 5; i++) {
      System.out.println("Iteration: " + i);
  }

Functions/Methods in Java

Description: Methods in Java are blocks of code that perform a specific task. They can take parameters and return values. The main method is the entry point for Java applications, but you can create additional methods to organize your code.


  // Defining a simple method
  public static void greet(String name) {
      System.out.println("Hello, " + name + "!");
  }
  
  // Calling the method
  greet("Larmel");  // Output: Hello, Larmel!

Object-Oriented Programming (OOP)

Description: Java is an object-oriented programming language, meaning that it uses classes and objects. Classes are blueprints for creating objects, and objects are instances of those classes. OOP concepts include inheritance, polymorphism, encapsulation, and abstraction.


  // Defining a class
  public class Person {
      String name;
      int age;
      
      // Constructor to initialize the object
      public Person(String name, int age) {
          this.name = name;
          this.age = age;
      }
      
      // Method to display the details
      public void display() {
          System.out.println("Name: " + name);
          System.out.println("Age: " + age);
      }
  }
  
  // Creating an object of the class
  Person person1 = new Person("Larmel", 19);
  person1.display();  // Output: Name: Larmel, Age: 19

Java Exception Handling

Description: Exception handling in Java allows you to deal with runtime errors, making your program more robust. You can use try, catch, and finally blocks to handle exceptions.


  // Handling exceptions with try-catch
  try {
      int result = 10 / 0;
  } catch (ArithmeticException e) {
      System.out.println("Error: Division by zero!");
  } finally {
      System.out.println("This block is always executed.");
  }
