Introduction to ArrayList

Description: ArrayList is a resizable array in Java. It is part of the java.util package and allows dynamic memory allocation, unlike regular arrays.

Creating an ArrayList

Description: Use the ArrayList class to create a dynamic list of elements.


  import java.util.ArrayList;
  
  public class Example {
    public static void main(String[] args) {
      ArrayList fruits = new ArrayList();
      fruits.add("Apple");
      fruits.add("Banana");
      fruits.add("Mango");
  
      System.out.println(fruits); // Output: [Apple, Banana, Mango]
    }
  }

Accessing Elements

Description: Use get(index) method to access elements from an ArrayList.


  ArrayList fruits = new ArrayList();
  fruits.add("Apple");
  fruits.add("Banana");
  
  System.out.println(fruits.get(0)); // Output: Apple

Modifying Elements

Description: Use set(index, element) to modify a specific element.


  ArrayList fruits = new ArrayList();
  fruits.add("Apple");
  fruits.set(0, "Orange");
  
  System.out.println(fruits); // Output: [Orange]

Removing Elements

Description: Use remove(index) or remove(object) to delete elements.


  ArrayList fruits = new ArrayList();
  fruits.add("Apple");
  fruits.add("Banana");
  
  fruits.remove("Apple"); // or fruits.remove(0);
  
  System.out.println(fruits); // Output: [Banana]

Looping Through an ArrayList

Description: You can use a for loop, for-each loop, or forEach method to iterate through elements.


  ArrayList fruits = new ArrayList();
  fruits.add("Apple");
  fruits.add("Banana");
  
  for (String fruit : fruits) {
    System.out.println(fruit);
  }

Common ArrayList Methods

Description: Useful methods in ArrayList include:

  • size() – Returns the number of elements
  • contains() – Checks if a value exists
  • clear() – Removes all elements
  • isEmpty() – Checks if list is empty

  ArrayList fruits = new ArrayList();
  fruits.add("Mango");
  
  System.out.println(fruits.size());       // 1
  System.out.println(fruits.contains("Mango")); // true
  fruits.clear();
  System.out.println(fruits.isEmpty());    // true
