Constructors in Java

Description: A constructor in Java is a special method that is called when an object of a class is created. It is used to initialize the object's state (i.e., its fields). Constructors have the same name as the class and do not have a return type, not even void.

Default Constructor

Description: A default constructor is provided by Java if no constructors are explicitly defined in the class. It initializes the object with default values (e.g., null for objects, 0 for numeric types, etc.).


  // Example of Default Constructor:
  public class Car {
      String brand;
      int year;
  
      // Default constructor
      public Car() {
          brand = "Unknown";
          year = 2000;
      }
  
      public void displayInfo() {
          System.out.println("Brand: " + brand + ", Year: " + year);
      }
  }
  
  public class Main {
      public static void main(String[] args) {
          Car car1 = new Car();  // Calls the default constructor
          car1.displayInfo();  // Output: Brand: Unknown, Year: 2000
      }
  }

Parameterized Constructor

Description: A parameterized constructor allows you to pass values to initialize the object's fields at the time of creation. This gives you more control over object initialization.


  // Example of Parameterized Constructor:
  public class Car {
      String brand;
      int year;
  
      // Parameterized constructor
      public Car(String brand, int year) {
          this.brand = brand;
          this.year = year;
      }
  
      public void displayInfo() {
          System.out.println("Brand: " + brand + ", Year: " + year);
      }
  }
  
  public class Main {
      public static void main(String[] args) {
          Car car1 = new Car("Toyota", 2020);  // Calls parameterized constructor
          car1.displayInfo();  // Output: Brand: Toyota, Year: 2020
      }
  }

Constructor Overloading

Description: Constructor overloading occurs when a class has multiple constructors with the same name but different parameter lists. This allows you to create objects in different ways, depending on the arguments passed to the constructor.


  // Example of Constructor Overloading:
  public class Car {
      String brand;
      int year;
  
      // Default constructor
      public Car() {
          brand = "Unknown";
          year = 2000;
      }
  
      // Parameterized constructor
      public Car(String brand, int year) {
          this.brand = brand;
          this.year = year;
      }
  
      public void displayInfo() {
          System.out.println("Brand: " + brand + ", Year: " + year);
      }
  }
  
  public class Main {
      public static void main(String[] args) {
          Car car1 = new Car();  // Calls default constructor
          car1.displayInfo();  // Output: Brand: Unknown, Year: 2000
  
          Car car2 = new Car("Honda", 2022);  // Calls parameterized constructor
          car2.displayInfo();  // Output: Brand: Honda, Year: 2022
      }
  }

The this Keyword in Constructors

Description: The this keyword refers to the current instance of the class. Inside a constructor, this is often used to differentiate between instance variables and parameters with the same name.


  // Example using the 'this' keyword:
  public class Car {
      String brand;
      int year;
  
      // Constructor with 'this' keyword
      public Car(String brand, int year) {
          this.brand = brand;  // 'this' refers to the instance variable
          this.year = year;
      }
  
      public void displayInfo() {
          System.out.println("Brand: " + brand + ", Year: " + year);
      }
  }
  
  public class Main {
      public static void main(String[] args) {
          Car car1 = new Car("Toyota", 2020);
          car1.displayInfo();  // Output: Brand: Toyota, Year: 2020
      }
  }

Constructor Chaining

Description: Constructor chaining is the process of calling one constructor from another constructor in the same class or from a superclass. This can be done using the this() keyword for constructors within the same class or super() for calling a superclass constructor.


  // Example of Constructor Chaining within the same class:
  public class Car {
      String brand;
      int year;
  
      // Default constructor
      public Car() {
          this("Unknown", 2000);  // Chaining to parameterized constructor
      }
  
      // Parameterized constructor
      public Car(String brand, int year) {
          this.brand = brand;
          this.year = year;
      }
  
      public void displayInfo() {
          System.out.println("Brand: " + brand + ", Year: " + year);
      }
  }
  
  public class Main {
      public static void main(String[] args) {
          Car car1 = new Car();  // Calls default constructor, which chains to parameterized constructor
          car1.displayInfo();  // Output: Brand: Unknown, Year: 2000
  
          Car car2 = new Car("Honda", 2022);  // Calls parameterized constructor
          car2.displayInfo();  // Output: Brand: Honda, Year: 2022
      }
  }
