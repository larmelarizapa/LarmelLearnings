Introduction to Fetch API

Description: The Fetch API provides a modern way to make HTTP requests in JavaScript. It allows you to request data from servers, and it returns a Promise that resolves to the response of the request.

Making a Basic GET Request

Description: A GET request is used to retrieve data from a server. The Fetch API makes it easy to perform GET requests and handle the responses.


  // Making a basic GET request to fetch data from an API
  fetch('https://api.example.com/data')
    .then(response => response.json())  // Parse the JSON response
    .then(data => console.log(data))    // Handle the response data
    .catch(error => console.log('Error:', error)); // Handle any errors

Handling JSON Response

Description: The Fetch API can easily handle responses in JSON format. Use the response.json() method to parse the response into a JavaScript object.


  // Fetching JSON data and handling the response
  fetch('https://api.example.com/user')
    .then(response => response.json())
    .then(user => {
      console.log(user.name);  // Access properties from the JSON response
      console.log(user.age);
    })
    .catch(error => console.log('Error:', error));

Making a POST Request

Description: POST requests are used to send data to a server. This can be done by including a body in the request.


  // Sending data with a POST request
  const data = {
    name: "Larmel",
    age: 19
  };
  
  fetch('https://api.example.com/user', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(data)  // Convert the JavaScript object to a JSON string
  })
    .then(response => response.json())
    .then(data => console.log(data))  // Handle the server response
    .catch(error => console.log('Error:', error));

Handling Errors with Fetch API

Description: The Fetch API does not throw an error for HTTP error status codes (like 404 or 500). You need to check the response.ok property to manually handle errors.


  // Handling errors by checking response status
  fetch('https://api.example.com/data')
    .then(response => {
      if (!response.ok) {
        throw new Error('Network response was not ok ' + response.statusText);
      }
      return response.json();
    })
    .then(data => console.log(data))
    .catch(error => console.log('Error:', error));  // Catch any error, including network issues

Using Async/Await with Fetch API

Description: You can use async and await to simplify Fetch API code, making it easier to read and write asynchronous code.


  // Using async/await for fetching data
  async function getUserData() {
    try {
      const response = await fetch('https://api.example.com/user');
      if (!response.ok) {
        throw new Error('Network response was not ok');
      }
      const user = await response.json();
      console.log(user);
    } catch (error) {
      console.log('Error:', error);
    }
  }
  
  getUserData();

Chaining Multiple Fetch Requests

Description: You can chain multiple Fetch requests to perform sequential operations, like retrieving data from different endpoints or combining results from multiple sources.


  // Chaining multiple requests to fetch user and their posts
  fetch('https://api.example.com/user')
    .then(response => response.json())
    .then(user => {
      console.log(user);
      return fetch(`https://api.example.com/posts/${user.id}`);
    })
    .then(response => response.json())
    .then(posts => console.log(posts))
    .catch(error => console.log('Error:', error));

Fetching Data from a JSON File

Description: You can also fetch data from local JSON files using the Fetch API. This is useful for loading configuration files, mock data, etc.


  // Fetching data from a local JSON file
  fetch('data.json')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.log('Error:', error));

Practical Example: Displaying Data from an API

Description: Below is a simple example of fetching data from an API and displaying it in a webpage.


  // Fetching and displaying data in the DOM
  async function displayUser() {
    const response = await fetch('https://api.example.com/user');
    const user = await response.json();
  
    const userDiv = document.getElementById('user');
    userDiv.innerHTML = `Name: ${user.name}, Age: ${user.age}`;
  }
  
  displayUser();
