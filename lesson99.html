Introduction to Local Storage

Description: Local Storage allows you to store data in the browser on a user's computer. This data persists even after the browser is closed, providing a way to save and retrieve data between sessions.

Storing Data in Local Storage

Description: You can store data in Local Storage using the localStorage.setItem() method. Data is stored as key-value pairs.


  // Storing a string in Local Storage
  localStorage.setItem('username', 'Larmel');
  
  // Storing a number
  localStorage.setItem('age', 19);

Retrieving Data from Local Storage

Description: To retrieve data, use the localStorage.getItem() method. If the item does not exist, it will return null.


  // Retrieving data from Local Storage
  let username = localStorage.getItem('username');
  console.log(username); // "Larmel"
  
  let age = localStorage.getItem('age');
  console.log(age); // "19"

Removing Data from Local Storage

Description: To remove an item from Local Storage, use the localStorage.removeItem() method.


  // Removing an item from Local Storage
  localStorage.removeItem('username');
  
  // Verify if it's removed
  let username = localStorage.getItem('username');
  console.log(username); // null

Clearing All Data in Local Storage

Description: If you want to remove all items from Local Storage, use the localStorage.clear() method.


  // Clearing all data from Local Storage
  localStorage.clear();
  
  // Verify that all data is cleared
  let age = localStorage.getItem('age');
  console.log(age); // null

Storing Objects in Local Storage

Description: Local Storage only stores strings, but you can store objects by converting them to a string using JSON.stringify().


  // Storing an object in Local Storage
  let person = { name: "Larmel", age: 19 };
  localStorage.setItem('person', JSON.stringify(person));
  
  // Retrieving and parsing the object
  let personData = JSON.parse(localStorage.getItem('person'));
  console.log(personData); // { name: "Larmel", age: 19 }

Checking the Size of Local Storage

Description: You can check how many items are currently stored in Local Storage using localStorage.length.


  // Checking the number of items in Local Storage
  console.log(localStorage.length); // Number of items

Using Local Storage in a Real-World Example

Description: Below is a practical example of how Local Storage can be used to save user preferences, such as theme settings.


  // Save the theme preference
  function setTheme(theme) {
    localStorage.setItem('theme', theme);
    document.body.className = theme;
  }
  
  // Load the theme preference when the page is loaded
  function loadTheme() {
    let theme = localStorage.getItem('theme');
    if (theme) {
      document.body.className = theme;
    }
  }
  
  // Example usage
  loadTheme();
  setTheme('dark'); // This will set the page to dark mode

Practical Example with Login

Description: Here's an example of using Local Storage to store a user's login status so that they remain logged in even after a page refresh.


  // Save login status
  function loginUser(username) {
    localStorage.setItem('loggedIn', true);
    localStorage.setItem('username', username);
  }
  
  // Check login status
  function checkLogin() {
    let loggedIn = localStorage.getItem('loggedIn');
    if (loggedIn) {
      let username = localStorage.getItem('username');
      console.log(`Welcome back, ${username}!`);
    } else {
      console.log('Please log in');
    }
  }
  
  // Example usage
  loginUser('Larmel');
  checkLogin(); // "Welcome back, Larmel!"
