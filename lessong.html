Methods in Java

Description: A method in Java is a block of code that performs a specific task. Methods are used to define behaviors for objects or perform certain operations that can be reused throughout a program.

Declaring a Method

Description: A method is declared by specifying the method's return type, name, and parameters (if any). A method can return a value or be void if it doesn't return anything.


  // Syntax:
  returnType methodName(parameters) {
      // method body
  }
  
  // Example:
  public static void greet() {
      System.out.println("Hello, welcome to Java!");
  }

Method Parameters

Description: Methods can accept parameters, which are values passed to the method when it is called. Parameters allow the method to operate on different inputs each time it's called.


  // Example:
  public static void greetPerson(String name) {
      System.out.println("Hello, " + name + "!");
  }
  
  greetPerson("Larmel");  // Output: Hello, Larmel!
  greetPerson("John");    // Output: Hello, John!

Method Return Type

Description: Methods can return a value. The return type defines the type of value the method will return. If the method does not return anything, the return type is void.


  // Example with return type:
  public static int addNumbers(int a, int b) {
      return a + b;
  }
  
  int result = addNumbers(5, 10);  // Output: 15
  System.out.println(result);  // Output: 15

Calling a Method

Description: A method can be called (or invoked) using its name, and if the method has parameters, appropriate values must be passed. Methods can be called from other methods in the same class or from other classes.


  // Calling a method from the same class:
  public static void main(String[] args) {
      greet();  // Calls the greet method
  }
  
  // Calling a method from another class:
  public class Calculator {
      public static int multiply(int a, int b) {
          return a * b;
      }
  }
  
  public class Main {
      public static void main(String[] args) {
          int result = Calculator.multiply(5, 3);  // Output: 15
          System.out.println(result);
      }
  }

Method Overloading

Description: Method overloading occurs when two or more methods have the same name but different parameters. It allows methods to perform similar tasks with different inputs.


  // Example of method overloading:
  public static void print(int number) {
      System.out.println(number);
  }
  
  public static void print(String message) {
      System.out.println(message);
  }
  
  public static void main(String[] args) {
      print(5);         // Output: 5
      print("Hello");   // Output: Hello
  }

Recursion

Description: Recursion is a technique where a method calls itself to solve a problem. It is often used for problems that can be broken down into smaller sub-problems.


  // Example of a recursive method to calculate factorial:
  public static int factorial(int n) {
      if (n == 0) {
          return 1;
      } else {
          return n * factorial(n - 1);  // Recursively calling factorial
      }
  }
  
  public static void main(String[] args) {
      System.out.println(factorial(5));  // Output: 120
  }

Method Scope

Description: Variables defined inside a method are local to that method and cannot be accessed outside it. This is called method scope. Local variables must be initialized before they are used.


  // Example:
  public static void calculateSum() {
      int sum = 0;  // Local variable
      for (int i = 1; i <= 5; i++) {
          sum += i;
      }
      System.out.println("Sum: " + sum);
  }
  
  public static void main(String[] args) {
      calculateSum();  // Output: Sum: 15
  }
