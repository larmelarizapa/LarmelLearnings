What is an Array?

Description: An array is a data structure used to store multiple values in a single variable.


let fruits = ["apple", "banana", "orange"];
console.log(fruits);         // ["apple", "banana", "orange"]
console.log(fruits[0]);      // "apple"
console.log(fruits.length);  // 3

Modifying Array Elements

Description: You can update or access specific items using their index.


let colors = ["red", "green", "blue"];
colors[1] = "yellow";
console.log(colors);  // ["red", "yellow", "blue"]

Adding and Removing Elements

Description: Use array methods like push, pop, shift, and unshift.


let numbers = [1, 2, 3];

numbers.push(4);       // Add to end
numbers.unshift(0);    // Add to start
numbers.pop();         // Remove from end
numbers.shift();       // Remove from start

console.log(numbers);  // [1, 2]

Looping Through Arrays

Description: You can use loops to access each item.


let names = ["Larmel", "Torres", "Arizapa"];

for (let i = 0; i < names.length; i++) {
  console.log(names[i]);
}

forEach Method

Description: A simpler way to loop through arrays.


let animals = ["dog", "cat", "rabbit"];

animals.forEach(function(animal) {
  console.log(animal);
});

Useful Array Methods

Description: Arrays come with many built-in methods.


let scores = [10, 20, 30, 40];

console.log(scores.includes(20)); // true
console.log(scores.indexOf(30));  // 2
console.log(scores.slice(1, 3));  // [20, 30]
console.log(scores.join(", "));  // "10, 20, 30, 40"

Arrow Functions

Description: A shorter syntax for writing functions.


const greet = (name) => {
  console.log("Hi, " + name);
};

greet("Larmel");
