Introduction to Spread and Rest Operators

Description: Both the spread operator (...) and the rest operator (...) are used to handle multiple elements in a more convenient and concise way. Although they look the same, their usage and purpose are different.

Spread Operator for Arrays

Description: The spread operator allows elements of an array to be expanded into individual elements. It can be used to copy or combine arrays, or pass individual array elements as function arguments.


  // Example of spreading an array
  let numbers = [1, 2, 3, 4];
  
  // Copying an array using spread
  let copiedNumbers = [...numbers];
  console.log(copiedNumbers); // [1, 2, 3, 4]
  
  // Combining two arrays
  let moreNumbers = [5, 6, 7];
  let combinedNumbers = [...numbers, ...moreNumbers];
  console.log(combinedNumbers); // [1, 2, 3, 4, 5, 6, 7]

Spread Operator for Objects

Description: The spread operator can also be used with objects to copy properties from one object to another or merge multiple objects.


  // Example of spreading an object
  let person = { name: "Larmel", age: 19 };
  
  // Copying an object using spread
  let copiedPerson = { ...person };
  console.log(copiedPerson); // { name: "Larmel", age: 19 }
  
  // Merging objects
  let additionalInfo = { city: "Gumaca", country: "Philippines" };
  let fullInfo = { ...person, ...additionalInfo };
  console.log(fullInfo); // { name: "Larmel", age: 19, city: "Gumaca", country: "Philippines" }

Rest Operator for Function Parameters

Description: The rest operator is used in function parameters to collect all remaining arguments into an array. It allows for more flexible function signatures.


  // Function using rest operator
  function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
  }
  
  console.log(sum(1, 2, 3, 4)); // 10
  console.log(sum(5, 6)); // 11

Rest Operator for Arrays

Description: The rest operator can be used in array destructuring to capture the remaining elements into an array.


  // Example using rest operator in array destructuring
  let [first, second, ...rest] = [1, 2, 3, 4, 5];
  
  console.log(first);  // 1
  console.log(second); // 2
  console.log(rest);   // [3, 4, 5]

Rest Operator for Objects

Description: The rest operator can also be used in object destructuring to capture the remaining properties into a new object.


  // Example using rest operator in object destructuring
  let person = { name: "Larmel", age: 19, city: "Gumaca" };
  let { name, ...otherInfo } = person;
  
  console.log(name);      // "Larmel"
  console.log(otherInfo); // { age: 19, city: "Gumaca" }

Differences Between Spread and Rest Operators

Description: While both the spread and rest operators look the same, their functionality is different:

  • Spread: Used to expand or spread elements or properties. It’s used when copying or merging arrays and objects, or when passing arguments to a function.
  • Rest: Used to collect remaining elements into an array or object. It’s used in function parameters or destructuring to collect multiple elements into a single variable.

Using Spread and Rest in a Real-World Example

Description: Combining the spread and rest operators can simplify complex operations, such as handling dynamic arguments and updating object properties.


  // Example combining both spread and rest
  function updateUserInfo(user, ...newInfo) {
    return { ...user, ...newInfo };
  }
  
  let user = { name: "Larmel", age: 19 };
  let updatedUser = updateUserInfo(user, { city: "Gumaca" });
  
  console.log(updatedUser); // { name: "Larmel", age: 19, city: "Gumaca" }
