Variables and Data Types in Java

Description: In Java, variables are used to store data values. Java is a statically typed language, meaning you must specify the type of data a variable will hold when you declare it. There are several data types available in Java, each used for different kinds of values.

Declaring Variables

Description: In Java, you must declare a variable before using it. When declaring a variable, you specify its type and its name. For example:


  // Declaring a variable
  int age;  // Declares an integer variable named 'age'
  
  // Initializing a variable
  age = 25;  // Assigning a value to 'age'
  
  // You can also declare and initialize the variable in one line:
  int height = 170;  // Declares and initializes 'height' as an integer

Primitive Data Types

Description: Java has 8 primitive data types: byte, short, int, long, float, double, char, and boolean. These are the building blocks for more complex data types. Letâ€™s look at each one:


  // Integer types
  byte a = 100;        // 8-bit integer
  short b = 1500;      // 16-bit integer
  int c = 10000;       // 32-bit integer (most commonly used)
  long d = 100000L;    // 64-bit integer (append 'L' for long)
  
  // Floating point types
  float e = 10.5f;     // 32-bit floating point number (append 'f' for float)
  double f = 20.99;    // 64-bit floating point number (default type for floating point literals)
  
  // Character type
  char g = 'A';        // 16-bit Unicode character
  
  // Boolean type
  boolean h = true;    // true or false

Non-Primitive Data Types

Description: Non-primitive data types (also known as reference types) include classes, interfaces, and arrays. They store references to objects rather than actual data values.


  // Example of using an array (non-primitive data type)
  int[] numbers = {1, 2, 3, 4, 5};  // Declaring and initializing an array
  
  // Example of a class (non-primitive data type)
  class Person {
      String name;
      int age;
      
      public Person(String name, int age) {
          this.name = name;
          this.age = age;
      }
  }
  
  // Creating an object of the Person class
  Person person1 = new Person("Larmel", 19);

Type Casting in Java

Description: Type casting is the process of converting one data type into another. Java supports both implicit (automatic) and explicit (manual) type casting:


  // Implicit Casting (Widening): smaller types are automatically converted to larger types
  int intValue = 100;
  long longValue = intValue;  // No explicit cast needed
  
  // Explicit Casting (Narrowing): larger types must be manually converted to smaller types
  long bigValue = 100000L;
  int smallValue = (int) bigValue;  // Casting required
  
  // Example with float to int conversion
  float floatValue = 10.75f;
  int intFromFloat = (int) floatValue;  // Result: 10 (fraction part is discarded)

String Data Type

Description: The String class is used to represent a sequence of characters. Unlike primitive data types, strings are non-primitive objects. You can create a string using double quotes:


  // Declaring a string variable
  String name = "Larmel";
  
  // String concatenation
  String greeting = "Hello, " + name + "!";  // Concatenate strings
  System.out.println(greeting);  // Output: Hello, Larmel!

Constants in Java

Description: Constants are variables whose values cannot be changed once assigned. In Java, constants are declared using the final keyword.


  // Declaring a constant
  final double PI = 3.14159;  // PI will always hold this value
  
  // Trying to change the value of PI will result in a compile-time error
  // PI = 3.14;  // Uncommenting this line will cause an error
