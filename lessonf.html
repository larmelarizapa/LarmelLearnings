Arrays in Java

Description: An array is a collection of variables of the same type, stored in a contiguous memory location. It allows you to store multiple values in a single variable, making it easier to manage collections of data.

Declaring and Initializing Arrays

Description: In Java, arrays can be declared and initialized in different ways. An array must have a fixed size, which is set when the array is created.


  // Declaring an array:
  int[] numbers; 
  
  // Initializing an array:
  numbers = new int[5];  // Array of size 5
  
  // Declaring and initializing an array in one step:
  int[] ages = {18, 19, 20, 21, 22};  // Array with 5 elements

Accessing Array Elements

Description: You can access array elements using their index. Array indices start from 0.


  int[] numbers = {10, 20, 30, 40, 50};
  System.out.println(numbers[0]);  // Output: 10
  System.out.println(numbers[3]);  // Output: 40

Modifying Array Elements

Description: Array elements can be modified by referencing their index and assigning new values.


  int[] numbers = {10, 20, 30, 40, 50};
  numbers[2] = 100;  // Changing the third element
  System.out.println(numbers[2]);  // Output: 100

Array Length

Description: The length of an array is a built-in property that returns the number of elements in the array. It can be accessed using array.length.


  int[] numbers = {10, 20, 30, 40, 50};
  System.out.println("Array length: " + numbers.length);  // Output: 5

Iterating Through Arrays

Description: You can use loops to iterate through all the elements in an array.


  // Using a for loop to iterate through an array:
  int[] numbers = {10, 20, 30, 40, 50};
  for (int i = 0; i < numbers.length; i++) {
      System.out.println(numbers[i]);  // Output: 10, 20, 30, 40, 50
  }
  
  // Using an enhanced for loop (for-each loop):
  for (int num : numbers) {
      System.out.println(num);  // Output: 10, 20, 30, 40, 50
  }

Multidimensional Arrays

Description: A multidimensional array is an array of arrays. It allows you to represent more complex data structures, like matrices.


  // Declaring a 2D array (array of arrays):
  int[][] matrix = {
      {1, 2, 3},
      {4, 5, 6},
      {7, 8, 9}
  };
  
  // Accessing an element in a 2D array:
  System.out.println(matrix[1][2]);  // Output: 6

Arrays and Methods

Description: You can pass arrays to methods as arguments and return them from methods.


  // Passing an array to a method:
  public static void printArray(int[] arr) {
      for (int i = 0; i < arr.length; i++) {
          System.out.println(arr[i]);
      }
  }
  
  int[] numbers = {10, 20, 30, 40};
  printArray(numbers);  // Output: 10, 20, 30, 40
  
  // Returning an array from a method:
  public static int[] createArray() {
      return new int[] {1, 2, 3, 4, 5};
  }
  
  int[] newArray = createArray();
  System.out.println(newArray[0]);  // Output: 1
