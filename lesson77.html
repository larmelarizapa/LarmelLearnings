Introduction to Destructuring Assignment

Description: Destructuring assignment allows you to unpack values from arrays or properties from objects into distinct variables. This feature makes your code cleaner and more concise.

Destructuring Arrays

Description: You can destructure arrays to extract individual elements into variables.


  // Array destructuring
  let numbers = [1, 2, 3, 4];
  
  let [first, second] = numbers;
  console.log(first);  // 1
  console.log(second); // 2
  
  // Skipping elements
  let [, , third] = numbers;
  console.log(third); // 3

Destructuring Objects

Description: You can destructure objects to assign their properties to variables. The variable names must match the property names.


  // Object destructuring
  let person = { name: "Larmel", age: 19 };
  
  let { name, age } = person;
  console.log(name); // "Larmel"
  console.log(age);  // 19
  
  // Renaming variables during destructuring
  let { name: fullName, age: years } = person;
  console.log(fullName); // "Larmel"
  console.log(years);    // 19

Default Values in Destructuring

Description: You can assign default values to variables in case the value from the array or object is undefined.


  // Array with default value
  let numbers = [1, 2];
  let [first, second, third = 3] = numbers;
  console.log(third); // 3
  
  // Object with default value
  let person = { name: "Larmel" };
  let { name, age = 19 } = person;
  console.log(age); // 19

Nested Destructuring

Description: Destructuring can also be done on nested arrays and objects, allowing you to unpack deeper values.


  // Nested array destructuring
  let numbers = [1, [2, 3], 4];
  let [, [second, third]] = numbers;
  console.log(second); // 2
  console.log(third);  // 3
  
  // Nested object destructuring
  let person = { name: "Larmel", address: { city: "Gumaca", barangay: "Binambang" } };
  let { address: { city, barangay } } = person;
  console.log(city);     // "Gumaca"
  console.log(barangay); // "Binambang"

Destructuring in Function Parameters

Description: You can destructure values directly in function parameters to make it easier to work with objects and arrays passed to functions.


  // Function with array destructuring
  function sum([a, b]) {
    return a + b;
  }
  console.log(sum([1, 2])); // 3
  
  // Function with object destructuring
  function greet({ name, age }) {
    console.log(`Hello, ${name}. You are ${age} years old.`);
  }
  greet({ name: "Larmel", age: 19 }); // "Hello, Larmel. You are 19 years old."

Destructuring with Rest Syntax

Description: The rest syntax can be used to collect the remaining elements of an array or properties of an object after destructuring.


  // Rest with array destructuring
  let [first, ...rest] = [1, 2, 3, 4, 5];
  console.log(first);  // 1
  console.log(rest);   // [2, 3, 4, 5]
  
  // Rest with object destructuring
  let { name, ...otherDetails } = { name: "Larmel", age: 19, city: "Gumaca" };
  console.log(name);           // "Larmel"
  console.log(otherDetails);   // { age: 19, city: "Gumaca" }
