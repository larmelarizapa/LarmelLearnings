What is the DOM?

Description: DOM (Document Object Model) represents the structure of your webpage as objects that JavaScript can manipulate.



<p id="myPara">Hello DOM!</p>
    

// JavaScript
let para = document.getElementById("myPara");
console.log(para.textContent); // Hello DOM!

Selecting Elements

Description: Use selectors to access HTML elements.


document.getElementById("id")
document.querySelector(".class")
document.querySelectorAll("tag")
    

let title = document.querySelector("h1");
let allParagraphs = document.querySelectorAll("p");

Changing Content

Description: Modify text and HTML using JavaScript.


let heading = document.getElementById("title");
heading.textContent = "Updated Title";
heading.innerHTML = "<em>Hello World</em>";

Changing Styles

Description: Use JavaScript to style elements dynamically.


let box = document.querySelector(".box");
box.style.backgroundColor = "lightblue";
box.style.fontSize = "20px";

Working with Attributes

Description: Get, set, and remove attributes from elements.


let link = document.querySelector("a");
link.setAttribute("href", "https://example.com");
console.log(link.getAttribute("href")); // "https://example.com"
link.removeAttribute("target");
