Loops in Java

Description: Loops are used to execute a block of code repeatedly based on a condition. In Java, there are several types of loops, including for, while, do-while, and enhanced for loop. Loops allow you to repeat tasks efficiently without writing repetitive code.

for Loop

Description: The for loop is used when the number of iterations is known beforehand. It consists of three parts: initialization, condition, and increment/decrement.


  // Syntax:
  for (initialization; condition; increment/decrement) {
      // code to be executed
  }
  
  for (int i = 0; i < 5; i++) {
      System.out.println("i = " + i);  // Output: i = 0, i = 1, i = 2, i = 3, i = 4
  }

while Loop

Description: The while loop is used when the number of iterations is not known beforehand and depends on a condition. The loop continues to execute as long as the condition is true.


  // Syntax:
  while (condition) {
      // code to be executed
  }
  
  int count = 0;
  while (count < 5) {
      System.out.println("count = " + count);  // Output: count = 0, count = 1, count = 2, count = 3, count = 4
      count++;
  }

do-while Loop

Description: The do-while loop is similar to the while loop, but it guarantees at least one iteration of the loop, even if the condition is false.


  // Syntax:
  do {
      // code to be executed
  } while (condition);
  
  int count = 0;
  do {
      System.out.println("count = " + count);  // Output: count = 0, count = 1, count = 2, count = 3, count = 4
      count++;
  } while (count < 5);

Enhanced for Loop (for-each loop)

Description: The enhanced for loop is used to iterate over arrays or collections. It's a simplified version of the for loop, eliminating the need for index manipulation.


  // Syntax:
  for (datatype element : collection) {
      // code to be executed
  }
  
  int[] numbers = {1, 2, 3, 4, 5};
  for (int num : numbers) {
      System.out.println(num);  // Output: 1, 2, 3, 4, 5
  }

break Statement

Description: The break statement is used to exit the loop prematurely, i.e., when a specific condition is met.


  // Syntax:
  for (int i = 0; i < 10; i++) {
      if (i == 5) {
          break;  // Exit the loop when i equals 5
      }
      System.out.println("i = " + i);  // Output: i = 0, i = 1, i = 2, i = 3, i = 4
  }

continue Statement

Description: The continue statement is used to skip the current iteration of the loop and proceed with the next iteration. It does not terminate the loop.


  // Syntax:
  for (int i = 0; i < 5; i++) {
      if (i == 2) {
          continue;  // Skip the iteration when i equals 2
      }
      System.out.println("i = " + i);  // Output: i = 0, i = 1, i = 3, i = 4
  }
